# Advanced Metal Calculator

![Metal Calculator](https://img.shields.io/badge/Metal%20Calculator-v1.0.0-1A4D2E)
![React](https://img.shields.io/badge/React-18.x-61DAFB)
![License](https://img.shields.io/badge/License-MIT-blue)

A comprehensive web application for calculating the weight and cost of various metal products. Designed for metal fabricators, engineers, and construction professionals who need quick and accurate calculations for estimating material costs.


## 📊 Current Functionality

- **Metal Weight Calculation**: Accurately calculate weight based on dimensions and material density
- **Cost Estimation**: Calculate costs based on weight and price per kg
- **Material Selection**: Choose from different metal types (steel, aluminum, etc.)
- **Unit Conversion**: Work in metric (mm) or imperial (inches) units
- **Project Organization**: Group calculations into projects for better organization
- **Responsive UI**: Clean, intuitive interface with material design principles
- **Localization**: Complete translation system for English and Bosnian languages

## 🛠️ Installation and Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/advanced-metal-calculator.git

# Navigate to the project directory
cd advanced-metal-calculator

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 🧪 Testing
